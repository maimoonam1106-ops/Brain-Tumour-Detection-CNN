# Brain Tumor Detection using CNN

This project uses a Convolutional Neural Network (CNN) to detect brain tumors from MRI images.  
The model is trained using deep learning techniques to classify whether a tumor is present or not.

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Streamlit
- NumPy

## Project Features
- MRI image classification
- Deep learning CNN model
- Streamlit web interface for prediction
- Image preprocessing and model prediction

## How to Run the Project

1. Install required libraries

pip install tensorflow opencv-python streamlit numpy pillow

2. Run the Streamlit app

streamlit run streamlit_app.py

## Dataset
The dataset used for this project contains MRI images of brain tumors.  
Due to large size, the dataset is not uploaded to this repository.

You can download similar dataset from Kaggle:
https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection
